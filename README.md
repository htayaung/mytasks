# MyTasks

## Background
A simple project to manage daily tasks.

This is a demo project using the following technologies.

## Technologies used
* C#
* .NET 7
* Blazor Server
* Entity Framework Core
* SQL Express
* Bootstrap
* [MudBlazor](https://mudblazor.com)
